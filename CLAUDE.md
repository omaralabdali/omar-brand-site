# Omar's Personal Brand Site

Dark green theme. Built with Next.js 16, Tailwind v4, Framer Motion.

## Colors
- bg: #080f0b
- surface: #0c1a13
- border: #1a3327
- accent: #14452f
- highlight: #39ff14
- text: #e8f5e9
- muted: #8aaa9a

## Fonts
- Display/Headings: Space Mono
- Body: DM Sans

## Structure
- src/components/ — shared components (Nav, CursorGlow, ScanlineOverlay)
- src/components/sections/ — page sections
- src/app/page.tsx — server component, imports all sections

## Commands
- `npm run dev` — start dev server
- `npm run build` — production build
